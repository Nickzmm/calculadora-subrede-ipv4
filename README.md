Calculadora de Sub-redes IPv4
Ferramenta desenvolvida em Java para automatizar o cálculo de parâmetros de endereçamento de rede. O objetivo do projeto foi aplicar de forma prática conceitos de infraestrutura e lógica de programação utilizando operadores bitwise.

Funcionamento Técnico
A aplicação realiza a manipulação de endereços IP e máscaras de sub-rede em nível binário. A partir da entrada de um IP e um prefixo CIDR, o sistema processa os bits para definir as fronteiras da rede. O desenvolvimento foi realizado no ambiente IntelliJ.

Principais Funções
Cálculo da máscara de sub-rede em formato decimal.

Identificação exata dos endereços de rede e de broadcast.

Definição do intervalo de IPs utilizáveis, do primeiro ao último host.

Cálculo da capacidade total de hosts disponíveis na sub-rede.

Execução
Compile o arquivo CalculadoraSubrede.java.

Execute a classe através do terminal.

Insira os dados solicitados (IP e prefixo) para visualizar os resultados.
